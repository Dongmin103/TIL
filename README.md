# Medi Log - 나만의 약물 복용 다이어리

## 🚀 서버 실행 방법

### Python 서버 (권장)
```bash
cd medi-log-enhanced
python3 server.py
```

### Node.js 서버
```bash
cd medi-log-enhanced
node server.js
```

### 간단한 HTTP 서버 (Python 내장)
```bash
cd medi-log-enhanced
python3 -m http.server 8080
```

## 🌐 접속 방법

서버 실행 후 다음 주소로 접속:

- **로컬 접속**: http://localhost:8080
- **네트워크 접속**: http://[IP주소]:8080 (서버 실행 시 표시됨)

## 📱 주요 기능

### ✨ 핵심 기능
- 🏠 **대시보드**: 오늘의 복용 현황과 날씨 기반 복용 상태 표시
- 📷 **처방전 스캔**: 처방전 촬영 및 자동 약물 인식
- 📊 **복용 분석**: 주간/월간 복용률 분석 및 AI 인사이트
- ⚙️ **설정**: 접근성 설정 및 건강 정보 관리

### 🎯 특별 기능
- ☀️⛅🌧️ **날씨 기반 복용 상태**: 복용률에 따른 직관적 시각화
- 📈 **월별 트렌드 차트**: 1-31일 꺾은선 그래프
- 💊 **약물 정보**: 상세한 약물 정보 및 부작용 안내
- 🔊 **음성 읽기**: 고령자를 위한 접근성 기능
- 📚 **처방 이력**: 과거 처방전 아카이브

### 👥 사용자 친화적 설계
- 📱 **모바일 최적화**: 414px 기준 반응형 디자인
- ♿ **접근성**: 큰 글씨, 고대비, 음성 읽기 지원
- 👴👵 **고령자 친화**: 65세 이상 자동 설정 적용

## 🛠️ 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Python/Node.js HTTP 서버
- **Storage**: LocalStorage (클라이언트 사이드)
- **Charts**: SVG 기반 커스텀 차트

## 📋 사용 가이드

1. **회원가입**: 기본 정보, 건강 정보, 처방 정보 입력
2. **로그인**: 전화번호와 이름으로 간편 로그인
3. **약물 관리**: 복용 완료 체크 및 상태 모니터링
4. **분석 확인**: 복용 패턴 분석 및 개선점 확인
5. **설정 조정**: 개인 맞춤 접근성 설정

## 🔧 문제 해결

### 서버가 시작되지 않는 경우
- 포트 8080이 사용 중인지 확인
- Python 3.x 또는 Node.js가 설치되어 있는지 확인

### 다른 기기에서 접속되지 않는 경우
- 같은 Wi-Fi 네트워크에 연결되어 있는지 확인
- 방화벽 설정 확인
- 서버 실행 시 표시되는 IP 주소 사용

### 화면이 깨져 보이는 경우
- 브라우저 캐시 삭제 (Ctrl+F5 또는 Cmd+Shift+R)
- 최신 브라우저 사용 권장

## 📞 지원

문제가 발생하면 브라우저 개발자 도구(F12)의 콘솔을 확인하여 오류 메시지를 확인해주세요.
