# 💊 Medi Log - 나만의 약물 복용 다이어리

개인 맞춤형 약물 관리 및 복용 기록을 위한 사용자 친화적인 웹 애플리케이션입니다.

## 🌐 온라인 데모
[Medi Log 체험하기](https://your-username.github.io/medi-log/)

## ✨ 주요 기능

### 📱 스마트 복용 관리
- 원클릭 복용 완료 처리
- 시간대별 자동 약물 표시
- 실시간 복용률 추적

### 📷 처방전 분석
- OCR 기반 처방전 자동 인식
- 약물 상호작용 검사
- AI 기반 복용 스케줄 최적화

### 📊 상세 분석
- 주간/월간 복용 현황 차트
- 개인화된 AI 인사이트
- 의료진과 데이터 공유

### ♿ 접근성 지원
- 큰 글씨 모드 (65세 이상 자동 적용)
- 음성 읽기 기능
- 고대비 모드
- 간단한 인터페이스 옵션

## 🎯 사용법

1. **홈 화면**: 현재 복용할 약물 확인 및 복용완료 버튼 클릭
2. **처방전 탭**: 사진 업로드 또는 "데모 분석" 실행
3. **분석 탭**: 복용 패턴 분석 및 "예시화면 보기" 확인
4. **설정 탭**: 건강정보 입력 및 접근성 기능 설정

## 🔧 기술 스택

- **Frontend**: Vanilla JavaScript, HTML5, CSS3
- **디자인**: 반응형 웹 디자인, 모바일 최적화
- **접근성**: WCAG 2.1 가이드라인 준수
- **브라우저 지원**: Chrome, Safari, Firefox, Edge
